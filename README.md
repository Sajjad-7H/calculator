# Advanced Calculator

A fully-featured, single-file calculator built with pure HTML, CSS, and JavaScript — no dependencies, no frameworks.

## Features

| Mode | Calculations |
|------|-------------|
| **Basic** | Add, subtract, multiply, divide, percentage, sign toggle |
| **Scientific** | sin, cos, tan (degrees), log, ln, √, x², x³, 1/x, \|x\|, π, e, n!, xʸ, 10ˣ |
| **Convert** | 10 categories: Length, Weight, Temperature, Area, Volume, Speed, Time, Data, Energy, Pressure |
| **History** | Scrollable log of all calculations |

## Usage

Just open `index.html` in any browser — no server needed.

```
open index.html
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Number input |
| `+ - * /` | Operators |
| `Enter` or `=` | Calculate |
| `Backspace` | Delete last digit |
| `Escape` | Clear all |
| `%` | Percentage |

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your calculator will be live at `https://yourusername.github.io/repo-name`
